参数模板。
每个游戏所需的配置参数都不同，这个json提供了游戏所需的参数，游戏的配置页面根据这个json生成对应的表单。
-----json下标含义-----
数字下标对应的是一个插件参数，
英文字符串对应的是参数组（参数组内包含多个插件参数），
下划线（_）对应的是冗余参数（groupNameMap对应参数组名称配置）。
-----一个插件参数内部配置-----
cName		参数名称
cAliasName	对应的英文名称
cDataType	参数类型(num:数字，string：字符串，date：日期时间，image：图片，radio：单选，		  checkbox：多选，prize：奖品)
cDataExt	radio/checkbox扩展数据(v:对应的值，n:对应的文字)
cDesc		参数描述
*/

开发环境http://lxb.plugin.dev.acewill.net 生产环境http://wg.welcrm.com

  • 请求URL

/activity/conf

  • 请求方式

POST

  • 请求参数
请求参数必填项参数类型参数说明
paid是Int活动ID
paName是String名称
paDesc是String描述
conf是Array配置
paPreviewCardList否String卡号白名单
  • 返回
返回参数参数类型参数说明
errcodeInteger错误码；0:成功；非0:失败
errmsgString执行结果信息
dataString活动详情
  • data数据
返回参数参数类型参数说明
paidInteger活动ID
  • 返回示例
 
xxxxxxxxxx
{
    "errcode":0,
    "errmsg":"success",
    "data":{
        "paid":"116"
    }
}
